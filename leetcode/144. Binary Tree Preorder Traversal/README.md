# 144. Binary Tree Preorder Traversal
Given the root of a binary tree, return the preorder traversal of its nodes' values.

## Example 1:
![inorder_1](https://user-images.githubusercontent.com/101626995/184601879-661148cf-3eed-459f-ad0c-79a961f428db.jpg)
```
Input: root = [1,null,2,3]
Output: [1,3,2]
```

## Example 2:
```
Input: root = []
Output: []
```

## Example 3:
```
Input: root = [1]
Output: [1]
```

## Constraints:
- The number of nodes in the tree is in the range [0, 100].
- -100 <= Node.val <= 100
